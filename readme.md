# PixiJS Interactive Particle Scene

This project is an interactive PixiJS scene built as part of a technical evaluation task.  
It demonstrates player movement, particle effects, pointer interactions, filters, masking, and performance handling using PixiJS.

---

## üöÄ Features

### Core Features
- Responsive PixiJS canvas (auto resize with window)
- Scrolling background using `TilingSprite`
- Controllable character using keyboard (Arrow keys / WASD)
- Character flips based on movement direction
- Character movement restricted within screen bounds
- Particle trail effect when character moves
- Click / tap explosion effect at pointer position
- Dynamic UI text showing click count

### Advanced Features
- Blur filter applied to the character and animated on movement
- Circular masking effect that follows the character
- Performance optimization with particle limits and cleanup
- Touch and mouse input support using pointer events
